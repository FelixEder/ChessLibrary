boolean Game.Board.isValidMove ( Piece  piece,
int  finalX,
int  finalY 
)

A function that determines whether a move is valid for any piece based on the following: 1) the move is within the board's bounds, 2) the move does not involve invalid leaping, 3) the move is not the same as the start location, 4) the move coordinates do not land on an ally piece, and 5) that the player's king is not in check (meaning the player has to move the king).
